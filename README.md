# Mixduino32
Graphical programming interface for ESP32 and ESPDUINO32,
Encontre  Projetos, exemplos e tutoriais para MIxduino32_IDE com ESPDUINO32 em  https://professorfarley.blogspot.com,
A versão atual é v1.1,
Mixduino32 é baseado em Mixly IDE, sendo traduzido para o português e  adaptado para  uso com placas baseadas em ESP32,
Mixduino32 é ideal para  projetos de física, robótia e internet das coisas com  ESDUINO-32
